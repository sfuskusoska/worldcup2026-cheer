# W杯2026 応援ゲーム

FIFAワールドカップ2026 ノックアウトステージのトーナメント表・優勝オッズに、
**5人（岡・大・田・片・南）が応援チームを決めて勝ち上がりを全員で楽しむ**1ファイル完結のWebアプリです。

- 応援チームは各自1チーム。🎡回転ルーレット（スタート/ストップ）か手動で決定（**重複なし**）
- 終了済みの試合の2チーム（南アフリカ・カナダ）は抽選対象外
- 選んだ内容は Supabase に保存され、同じページを開いた全員にリアルタイム反映
- トーナメント表に各自の応援チームを色付きマークで表示
- 「応援チームの勝ち上がり」リーダーボード（進出 / 敗退 / 優勝を自動判定）
- 「🔄 最新結果で更新」で公開スコア（ESPN JSON）から勝者を次ラウンドへ反映
- スマホ最適化（縦1カラム、全体/スクロール/詳細表示）

## ホスティング
GitHub Pages（静的ホスティング）。`index.html` がそのままトップページです。

## データ共有先（Supabase）
共有を有効にするには `index.html` 内の以下を設定します（未設定でも端末内保存で動作）。

```js
const SUPABASE_URL = "https://xxxx.supabase.co";
const SUPABASE_ANON_KEY = "ey...";
```

セットアップ手順とSQLは [`SUPABASE_SETUP.md`](./SUPABASE_SETUP.md) を参照。

## 注意
- `anon` キーは公開用キーです。ページに埋め込んでも問題ありません。
- 共有テーブルは「URLを知る5人で使う」前提のオープン設定です（低リスク用途）。
- オッズは参考値で、賭けの推奨ではありません。

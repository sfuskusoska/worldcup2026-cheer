# W杯2026 応援ゲーム

FIFAワールドカップ2026 ノックアウトステージのトーナメント表・優勝オッズに、
**5人（岡・大・田・片・南）に応援チームを配分して、勝ち上がりを全員で楽しむ**1ファイル完結のWebアプリです。

- 応援チームはホストが `ASSIGNMENTS` に固定配分（30チームを5人へ各6チーム）
- 終了済みの試合の2チーム（南アフリカ・カナダ）は配分対象外
- 応援チームが試合に勝つたびに、持ち主へ **1勝 = 1ポイント**
- 近い試合は色分け表示（3日以内 / 24時間以内 / ライブ / 終了）
- 🎡回転ルーレット（スタート/ストップ）は遊び用として残し、配分には反映しない
- 試合前BGMをループ再生（開始・停止・ループ継ぎ目にフェード）
- トーナメント表に各自の応援チームを色付きマークで表示
- 「応援ポイント」リーダーボード（合計ポイント順）
- 「🔄 最新結果で更新」で公開スコア（ESPN JSON）から勝者を次ラウンドへ反映
- スマホ最適化（縦1カラム、全体/スクロール/詳細表示）

## ホスティング
GitHub Pages（静的ホスティング）。`index.html` がそのままトップページです。

## 配分の変更
`index.html` の `ASSIGNMENTS` を編集します。ページ内に固定表示されるため、リアルタイム入力や外部データベースは使いません。

## BGM
音源は `assets/audio/pre-match-bgm.mp3` です。ブラウザの自動再生制限に合わせ、画面右上の `♪ BGM` ボタンで再生/停止します。

## 注意
- オッズは参考値で、賭けの推奨ではありません。
